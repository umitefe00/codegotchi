# 🐾 CodeGotchi - Geliştirici Odaklı Dijital Evcil Hayvan

CodeGotchi, React Native ve Expo kullanılarak geliştirilmiş, "vibe coding" ve oyunlaştırma (gamification) prensiplerini barındıran bir dijital evcil hayvan simülasyonudur.

## 🚀 Özellikler & Oyunlaştırma
- **Dinamik State Yönetimi:** Açlık ve mutluluk değerlerine göre petin ruh hali anlık değişir.
- **Zamanlayıcı (Auto-Drain):** Evcil hayvanın açlık ve mutluluk değerleri zamanla kendiliğinden düşerek oyun zorluğu yaratır.
- **Seviye ve XP Sistemi:** Evcil hayvanınızı besledikçe ve oynadıkça XP kazanırsınız. 100 XP'de seviye atlanır.
- **Vibe Coding UI/UX:** Duruma göre değişen arka plan renkleri ve zıplama animasyonları (Animated.spring).
- **Başarımlar (Badges):** Belli seviyelere ulaşıldığında "🔥 Junior Dev" ve "🏆 Senior Pet" gibi rozetlerin kilidi açılır.
